(default=4096) Max size, in bytes, of AMQP data frames. Protocol max is 2^32-1. Actual value is negotiated with the server.

Optional heartbeat

heartbeat?: number
  • Defined in normalize.ts:32

(default=60) Period of time, in seconds, after which the TCP connection should be considered unreachable. Server may have its own max value, in which case the lowest of the two is used. A value of 0 will disable this feature. Heartbeats are sent every heartbeat / 2 seconds, so two missed heartbeats means the connection is dead.
